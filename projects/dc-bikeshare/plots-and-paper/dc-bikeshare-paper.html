Bikeshare Usage Analysis - Washington, D.C.

Tools used: Python, pandas, seaborn, matplotlib, statsmodels

View Notebook GitHub Repo

Objective

The goal of this project is to analyze bikeshare ridership patterns by distinguishing between casual riders and registered riders, then examining how behavior changes across working days, non-working days, and temperature conditions.

Dataset Overview

  • workingday: indicator for workdays
  • casual, registered, count: ride counts by user type
  • season, holiday, and date metadata

Before analysis, the data was cleaned and filtered to enable conditional comparisons and plotting.

Methods and Visualizations

1. Scatterplot of Casual vs. Registered Riders

A scatterplot compared casual and registered ride volumes across the full dataset and showed a positive but non-linear trend.

2. LOWESS Smoothing

LOWESS helped isolate the underlying temperature relationship with casual ridership, especially on weekends.

3. KDE Density Plot

KDE contours highlighted where casual and registered rider counts cluster most strongly on working days.

4. Overlay KDE Comparison

Overlay KDE plots showed that registered ridership dominates on working days while casual ridership peaks on non-working days.

5. Temperature Effects

Linear regression plots showed warmer temperatures strongly increase casual ridership, particularly on weekends.

6. Hourly Usage Patterns

Hourly averages revealed morning and evening commute peaks for registered riders and more midday activity for casual riders.

Key Insights

  • Registered riders are more consistent on working days, indicating commuting behavior.
  • Casual riders spike on weekends and holidays, suggesting recreational use.
  • Temperature is a strong driver of casual ridership.
  • KDE and LOWESS methods captured more nuance than basic scatterplots alone.

Conclusion

This analysis shows how different rider types engage with urban transit systems and suggests ways to improve marketing, infrastructure planning, and membership conversion strategies.
